Voice Features
Requirements for Voice Features

Voice recording and voice calls require:

  • Secure Connection (HTTPS) - Required for remote access
  • Microphone Permission - Allow browser access to your microphone
  • Modern Browser - Chrome, Firefox, Safari, or Edge
Troubleshooting

If you're accessing the site via HTTP, voice features won't work on remote networks.

Solutions:

  • Access the site using HTTPS instead
  • Use localhost for development
  • Set up SSL certificates on the server

If microphone access is denied:

  • Click the microphone icon in your browser's address bar
  • Select "Allow" for microphone access
  • Refresh the page after granting permission
  • Check if other apps are using the microphone
